# Contributing

## English

Keep the project conservative:

- Add only missing bookmarks.
- Do not delete, rename, or move existing bookmarks unless the behavior is explicitly opt-in.
- Keep tests synthetic and free of private URLs.
- Run `python3 -m unittest discover -s tests -v` before opening a pull request.

## 中文

请保持项目的保守策略：

- 只补齐缺失书签。
- 除非用户显式启用，否则不删除、不重命名、不移动现有书签。
- 测试数据必须是合成数据，不要包含私人 URL。
- 提交 pull request 前请运行 `python3 -m unittest discover -s tests -v`。
